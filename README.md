# GPT vs Claude

*"Hi, I'm a GPT." "And I'm a Claude."*

A web comic about two AI assistants with very different personalities. Same weird questions from users. Very different vibes.

**GPT** answers first, hedges never. **Claude** hedges first, answers eventually.
